About QueueFree

Queue control for operations, not just patients

QueueFree replaces manual token calling with structured, real-time queue control accessible to reception, doctors, and patients simultaneously. Every stakeholder sees the same live state — no phone calls, no guesswork, no delay blindness.

SaaS Subscription Model
No hardware. Monthly subscription. Deployable at any hospital reception within minutes.
Dual-App Architecture
Separate Patient App and Doctor App. Each scoped to its user. Both connected to a shared live backend.
Doctor Delay Transparency
Reception can log doctor delays in seconds. Patients receive updated estimates without manual communication.
SMS Fallback Ready
Token confirmations via SMS for clinics with limited in-building network coverage.

Live Queue Visibility

Reception, doctors, and patients all see the same queue state simultaneously. Token status updates propagate instantly across all devices — no polling, no manual refresh.

Socket.io • Real-time push

Predictive Wait-Time Estimation

Estimated wait is calculated deterministically — no machine learning required at this stage. The formula is transparent and adjustable per doctor.

Wait = (queue depth × avg. consult time) + doctor delay

Structured Priority Token Handling

Tokens carry a priority score. Pre-booked, elderly, and urgent cases surface ahead of standard walk-ins. Queue order is enforced systematically, not left to reception discretion.

Priority scoring • Rule-based
